# Firebase Authorized Domain Setup

## Current Deployment Domain
**Add this domain to Firebase:**

## How to Add Domain to Firebase

1. Go to Firebase Console:
   https://console.firebase.google.com/project/sweepy-863bf/authentication/settings

2. Scroll down to "Authorized domains" section

3. Click "Add domain"

4. Enter: `sweepy-hkkn1zngc-shreyashi-duttas-projects.vercel.app`

5. Click "Add"

## Alternative: Add Wildcard Domain (Recommended)

To avoid adding each new deployment URL, you can add:
- `*.vercel.app` (if Firebase allows wildcards)

Or add your custom domain if you have one configured in Vercel.
